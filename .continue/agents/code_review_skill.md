# Code Review Skill

## Description
Perform comprehensive code reviews to ensure code quality, security, and adherence to best practices. This skill identifies potential issues, suggests improvements, and ensures maintainability.

## Capabilities
- Code quality assessment
- Security vulnerability identification
- Performance optimization suggestions
- Best practice enforcement
- Bug detection and prevention

## Usage Examples
- "Review this Python function for potential security issues"
- "Analyze this JavaScript code for performance bottlenecks"
- "Check this Java class for adherence to coding standards"
- "Identify potential bugs in this C++ implementation"

## Skill Parameters
- review_type: Security, Performance, Style, Bug-Fixing
- language: Python, JavaScript, Java, C++, etc.
- complexity: Basic, Intermediate, Advanced
- focus_areas: Code Quality, Security, Performance, Maintainability
