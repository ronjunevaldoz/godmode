# Data Analysis Skill

## Description
Perform data analysis, generate insights, and create visualizations from datasets. This skill helps interpret data patterns, identify trends, and provide actionable recommendations based on analytical findings.

## Capabilities
- Statistical analysis and interpretation
- Data visualization generation
- Trend identification and forecasting
- Performance metrics analysis
- Comparative analysis between datasets

## Usage Examples
- "Analyze this sales dataset and identify key trends"
- "Generate a report on user engagement metrics"
- "Compare the performance of different marketing campaigns"
- "Identify outliers in this financial data set"

## Provider Configuration
- Primary: ollama_gemma4 (local)
- Fallback: claude_architect, codex_primary

## Skill Parameters
- analysis_type: Statistical, Trend, Comparative, Predictive
- data_format: CSV, JSON, Database, Excel
- visualization: Chart, Graph, Table, Dashboard
- insights_level: Basic, Detailed, Strategic
