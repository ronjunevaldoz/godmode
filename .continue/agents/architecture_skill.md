# System Architecture Skill

## Description
Design and review system architectures, including software architecture, database design, and infrastructure planning. This skill helps create scalable, maintainable, and efficient system designs.

## Capabilities
- Software architecture design (microservices, monoliths)
- Database schema design and optimization
- Infrastructure planning and cloud architecture
- System scalability analysis
- Security architecture review

## Usage Examples
- "Design a microservices architecture for an e-commerce platform"
- "Create a database schema for a social media application"
- "Review the current system architecture for performance bottlenecks"
- "Plan the infrastructure for a high-traffic web application"
