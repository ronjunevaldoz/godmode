# Documentation Skill

## Description
Create comprehensive technical documentation including API references, user guides, and system architecture documents. This skill generates clear, well-structured documentation that helps developers understand and use the system effectively.

## Capabilities
- API documentation generation (REST, GraphQL)
- User manuals and guides
- System architecture diagrams and explanations
- Code comments and docstrings
- Technical specifications and requirements

## Usage Examples
- "Generate API documentation for our user management service"
- "Create a user guide for the mobile application"
- "Document the database schema and relationships"
- "Write technical specifications for the authentication module"

## Provider Configuration
- Primary: ollama_llama3 (local)
- Fallback: claude_architect, gemini_vision

## Skill Parameters
- document_type: API, User Guide, Architecture, Code Comments
- audience: Developers, End Users, Technical Stakeholders
- format: Markdown, PDF, HTML
- scope: Module, System, Component
