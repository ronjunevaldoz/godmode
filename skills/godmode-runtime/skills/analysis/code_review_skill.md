# Code Review Skill

## Description
This skill enables the system to perform comprehensive code reviews. It analyzes code quality, identifies potential issues, and provides improvement suggestions based on best practices.

## Capabilities
- Code quality assessment
- Bug identification
- Security vulnerability detection
- Performance optimization suggestions
- Best practice compliance checking

## Usage Examples
- "Review this Python function for potential bugs and improvements"
- "Analyze the security aspects of this JavaScript code"
- "Perform a comprehensive code review on this Java class"

## Required Parameters
- Code: The code to be reviewed
- Language: Programming language of the code
- Focus Areas: Specific areas to focus on (security, performance, readability, etc.)
