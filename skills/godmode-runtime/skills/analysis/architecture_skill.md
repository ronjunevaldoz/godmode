# Architecture Skill

## Description
This skill enables the system to design, review, and analyze software architectures. It provides expertise in system design patterns, scalability considerations, and best practices for various architectural approaches.

## Capabilities
- System architecture design
- Architecture pattern recognition
- Scalability analysis
- Performance optimization recommendations
- Security consideration assessment

## Usage Examples
- "Design a microservices architecture for an e-commerce platform"
- "Review the current system architecture and suggest improvements"
- "Analyze the scalability of this distributed system design"

## Required Parameters
- System Type: Type of system being designed/reviewed (web, mobile, distributed, etc.)
- Requirements: Specific requirements and constraints
- Scale: Expected scale and performance requirements
