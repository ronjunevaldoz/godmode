# Analysis Skill

## Description
This skill enables the system to perform data analysis and generate insights from datasets. It can handle various types of analytical tasks including statistical analysis, trend identification, and data visualization recommendations.

## Capabilities
- Data analysis and interpretation
- Statistical pattern recognition
- Trend identification in datasets
- Data visualization recommendations
- Comparative analysis between datasets

## Usage Examples
- "Analyze this sales dataset and identify key trends"
- "Perform statistical analysis on the provided research data"
- "Identify patterns in user behavior from the collected metrics"

## Required Parameters
- Dataset: The data to be analyzed (can be CSV, JSON, or raw data)
- Analysis Type: Specific type of analysis requested (statistical, trend, comparative, etc.)
